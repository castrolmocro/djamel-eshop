# Djamel E Shop — السوق المحلي الجزائري

## 🎯 الهدف من الموقع

**Djamel E Shop** هو منصة سوق محلي جزائري تتيح للأشخاص والمتاجر والمصانع نشر منتجاتهم وخدماتهم وعرضها للمشترين القريبين من نفس المنطقة.

الفكرة الأساسية: **ابحث عن منتج → اجد بائعاً قريبك → تواصل مباشرة → أتم الصفقة**.

---

## 👥 من يمكنه استخدام الموقع؟

| النوع | الوصف |
|-------|-------|
| **أفراد** | بيع وشراء منتجات مستعملة أو جديدة |
| **متاجر** | عرض بضاعة المتجر للعملاء القريبين |
| **مصانع** | الوصول لعملاء المنطقة مباشرة |
| **مقدمو خدمات** | نشر خدماتهم (سباكة، كهرباء، دروس...) |

---

## 🌟 المميزات الرئيسية

### للمستخدمين:
- 🔍 **بحث جغرافي** — ابحث عن منتجات حسب الولاية والمدينة
- 📢 **نشر إعلانات** — أضف منتجاتك بسهولة مع صور وتفاصيل
- 💬 **مراسلة مباشرة** — تواصل مع البائع مباشرة من الموقع
- ⭐ **تقييمات** — قيّم البائعين والمنتجات
- 📦 **إدارة الطلبات** — تتبع مشترياتك ومبيعاتك

### تقني:
- 🌐 **ثلاث لغات** — العربية (افتراضي) + الفرنسية + الإنجليزية
- 🌙 **وضع مظلم/فاتح** — قابل للتبديل
- 📱 **متجاوب تماماً** — يعمل على الجوال والكمبيوتر
- 🔐 **مصادقة آمنة** — Clerk Auth
- 🗄️ **قاعدة بيانات** — PostgreSQL + Drizzle ORM

---

## 🏗️ هيكل المشروع

```
/artifacts
  /djamel-eshop      → الواجهة الأمامية (React + Vite + Tailwind)
  /api-server        → الخادم الخلفي (Express.js + TypeScript)
/lib
  /db                → قاعدة البيانات (Drizzle ORM + PostgreSQL)
  /api-spec          → OpenAPI spec
  /api-client-react  → React Query hooks (auto-generated)
  /api-zod           → Zod schemas (auto-generated)
```

## 🗄️ جداول قاعدة البيانات

| الجدول | الوصف |
|--------|-------|
| `profiles` | بروفيلات المستخدمين (اسم، صورة، ولاية، متجر؟) |
| `listings` | الإعلانات (عنوان، وصف، سعر، ولاية، نوع، صور) |
| `categories` | تصنيفات الإعلانات (12 تصنيف) |
| `conversations` | محادثات بين المشتري والبائع |
| `messages` | رسائل المحادثات |
| `orders` | الطلبات (حالة: معلق→مؤكد→مكتمل/ملغى) |
| `reviews` | تقييمات الإعلانات والبائعين |

## ⚙️ متغيرات البيئة المطلوبة

### Frontend (Railway / Vite):
```env
VITE_CLERK_PUBLISHABLE_KEY=pk_live_...
VITE_CLERK_PROXY_URL=https://yourdomain.com/clerk
VITE_API_URL=https://your-api.railway.app
```

### Backend (Railway / API Server):
```env
DATABASE_URL=postgresql://...
CLERK_SECRET_KEY=sk_live_...
PORT=3000
```

## 🛠️ تشغيل محلياً

```bash
# تثبيت الحزم
pnpm install

# تشغيل كل الخدمات
pnpm --filter @workspace/api-server dev
pnpm --filter @workspace/djamel-eshop dev

# ترحيل قاعدة البيانات
cd lib/db && pnpm db:push

# بذر التصنيفات
curl -X POST http://localhost:PORT/api/seed
```
